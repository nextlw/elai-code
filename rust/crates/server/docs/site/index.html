Documentação da API HTTP do elai-server

O elai-server é um servidor HTTP em Rust (Axum) que expõe orquestração de sessões de IA, operações de workspace, Git, ferramentas, MCP, plugins e telemetria — pensado para integrações locais via loopback com autenticação por Bearer token.

Base padrão: 127.0.0.1:8456 Auth: Bearer (token em ~/.elai/server-token) Streaming: SSE + WebSocket ~70 endpoints · 16 grupos
